# AI Portfolio

Modern, dark-themed portfolio built with React + Tailwind CSS.

Quick start

```bash
cd ai-portfolio
npm install
npm run dev
```

Features
- React + Vite
- Tailwind CSS
- Dark theme with glassmorphism
- Responsive layout and smooth scrolling
- React Three Fiber 3D hero scene
- Local image assets for projects and hero preview

Deployment
- Netlify: connect this repo and use `npm run build` with `dist` as the publish directory.
- Vercel: this repo is configured with `vercel.json` to build using `@vercel/static-build` and publish `dist`.

Customize contact email in `src/components/Contact.jsx`.
